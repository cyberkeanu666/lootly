1. System Overview

Lootly.gg is a next-generation, full-stack, provably fair sweepstakes and engagement platform designed for stream hosts and digital creators. It solves the critical double-spend trust problem in online giveaways by utilizing cryptographically verifiable seeds and mathematical drawing equations that can be audited by third parties.

3. Modules & Component Specifications

A. Landing Home View

Introduces streamers to core capabilities. Features an inline Host Sandbox Session portal where users can instantly log in as pre-seeded profiles to test campaign setups.

B. Creator Command Center Dashboard

Generates precise overview traffic cards (Direct visits, referrals, social referrals) and live conversion charts. Enables seamless creation, upgrade to Pro tier support, and giveaway link generation.

Public interface displaying full timers, rules, verified entrant lists, and pre-published SHA-256 seed transparency keys. Features referral invite widgets that inject cookies pointing to the referrer handles.

D. Live Drawing Dome

Streamer-friendly sweepstakes rendering wheel featuring interactive scrolling ribbons, countdowns, configurable velocities, winner logs, and particle celebrations.

4. Provably Fair Mathematics

Lootly ensures absolute fairness by using unshifting deterministic mathematical execution:

Winner Modular Equation
Index = BigInt("0x" + SHA256(RevealedSeed)) % BigInt(TotalTickets)
  1. Hosts input secret seeds behind a pre-published SHA-256 hash.
  2. Registered participant lists are padded according to earned invite weights and sorted alphabetically.
  3. When the seed is revealed, the host keys are converted into hex formats, combined, and calculated via BigInt modulo math to select the winner index in a 100% unbiased manner.

Simulated
